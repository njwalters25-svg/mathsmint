# MathsMint

A printable Functional Skills Maths Level 1 worksheet generator.

## Features

- Single-topic and mixed-topic worksheets
- Easy, medium, and hard difficulty
- Calculator, non-calculator, and mixed papers
- Varied real-world question formats
- Automatic answers and worked solutions
- Print-ready worksheet and answer-sheet layouts

## Run locally

Open `index.html` directly, or serve the folder:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy to Netlify

Import this repository into Netlify. No build command is required; the publish
directory is the repository root.
